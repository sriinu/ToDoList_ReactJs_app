# React Todo List

Simple React Todo App

<img src="public/app-pic.png" width="60%">

## Demo 
https://todo-vanila-react.vercel.app/

## Installation

```bash
npm install
```

or

```bash
yarn install
```

## Start

```bash
npm run dev
```
or
```bash
yarn dev
```

## Functionality
1. Adding, editing, toggling, deleting todo's
2. Filtering and deleting completed todo's
3. Theme switching
